# Short.er

## Descrição:
Shorter is an application designed to shorten Urls.
For example, the URL `https://google.com.br/search?q=shorter` can be shortened to `https://short.er/sDfgT5a`

## Requerimentos:

* Ruby 2.7.1
* Rails 6.0.3.1

### Getting started
* git clone git@github.com:leobagua/short.er.git
* cd folder
* bundle
* rails s
* http://localhost:3000
